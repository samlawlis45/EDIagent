# Roadmap

## Near Term

- Add first-class connectors for real EDI platforms (Cleo CIC, Stedi, SPS)
- Add workflow orchestration endpoint for multi-agent runs
- Add persistence layer for run history and audit events
- Add webhook eventing for agent lifecycle notifications

## Mid Term

- Add policy engine for approve/propose/execute modes
- Add role-based access control and tenant boundaries
- Add replay tooling and deterministic run traces

## Long Term

- Add distributed worker mode for high-volume workloads
- Add plugin marketplace for community adapters and agent packs
- Publish stable v1 API and compatibility policy
