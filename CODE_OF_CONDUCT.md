# Code of Conduct

## Our Standard

We are committed to a harassment-free and constructive community for everyone.
Participants are expected to communicate respectfully, focus on technical substance,
and collaborate in good faith.

Examples of expected behavior:

- Use welcoming and inclusive language
- Accept constructive feedback
- Focus on what is best for the project and users
- Show empathy toward other community members

Examples of unacceptable behavior:

- Harassment, discrimination, intimidation, or personal attacks
- Trolling, insulting or derogatory comments
- Publishing private information without explicit permission
- Any conduct that is inappropriate in a professional setting

## Scope

This Code of Conduct applies in project spaces (issues, pull requests, discussions,
documentation, and chat channels) and when representing the project publicly.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this policy.
Maintainers may remove, edit, or reject comments, commits, code, wiki edits,
issues, and other contributions that are not aligned with this Code of Conduct.

## Reporting

Report violations to: `security@ediagent.com`

All reports will be reviewed promptly and handled confidentially to the extent practical.
